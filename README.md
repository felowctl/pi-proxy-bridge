# Pi Proxy Bridge
Raspberry Pi as a WiFi-to-WiFi hotspot with transparent proxy by Xray.

> [!IMPORTANT]
> Requires two different wifi interfaces. Probably, you need a separate USB wifi dongle.

## Installation
```bash
git clone https://github.com/felowctl/pi-proxy-bridge
cd pi-proxy-bridge
chmod +x install.sh
sudo ./install.sh
```

Tested on Raspberry Pi 4B with TP-Link Archer TX1U Nano and [this driver](https://github.com/Kiborgik/aic8800dc-linux-patched).
